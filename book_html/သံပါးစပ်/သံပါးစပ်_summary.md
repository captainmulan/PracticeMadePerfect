## Writing Style

### Character Building (GOT Style)
- **Slow reveal**: Main character not obvious early (like Jon Snow in S1-2)
- **Multiple POVs**: Characters appear, grow, disappear
- **No plot armor**: Anyone can die
- **Ensemble cast**: Hero grows alongside others

### Structure
- **Main Chapter**: 3,000 words (1 hour read)
- **Subchapters**: 3,000 words each (30 min read)
- **Subchapters per Main**: 2-3
- **Total per Main Chapter**: 6,000-9,000 words

### Format
- **HTML only**: `<h3>`, `<p>`, `<blockquote>`, `<hr>`
- **Opening**: One sentence (saying/puzzle/quote)
- **Ending**: "Next - [preview]"
- **No Markdown, CSS, or extra spacing**

### Pacing
- **Slow build → Strong climax**
- **Rich narration (70-80%)**
- **Dialogue (20-30%)**
- **Ancient language, easy to read**

### Chapter Ending
- **Cliffhanger always**
- **Reason to continue**
