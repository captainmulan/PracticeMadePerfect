📖 What is a Loop?

A loop tells the computer to repeat something, again and again, without writing the same code over and over.

Simon plays back the whole sequence using a loop - one color at a time - and it gets longer every round!

💻 JavaScript Code

for (let i = 0; i < sequence.length; i++) { lightUp(sequence[i]); await wait(500); }
🌟 Did you know?

Loops are everywhere in code - playing a song list, animating a character, or checking every item in a list all use loops so programmers don't have to repeat themselves.
