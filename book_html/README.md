# Educational Ebooks for Kids

This directory contains interactive educational ebooks designed for children aged 7-8.

## Ocean Adventure

**Author:** PracticeMadePerfect  
**Target Age:** 7-8 years  
**Chapters:** 28

A comprehensive journey through the ocean, covering:
- Ocean zones (Sunlight, Twilight, Midnight, Abyss)
- Coral reefs and their ecosystems
- Marine mammals (whales, dolphins, seals)
- Ocean fish diversity
- Environmental themes and conservation

**Features:**
- Standalone HTML files with embedded CSS and JavaScript
- Ocean-themed dark blue gradient backgrounds with bubble animations
- Interactive clickable zones and creatures
- Interactive quizzes with immediate feedback and scoring
- Kid-friendly language and responsive design

**Chapter Structure:**
- 001-002: Introduction (Book Briefing, Index)
- 003-005: Ocean Overview (Overview, Explained, Quiz)
- 006-008: Sunlight Zone (Intro, Explained, Quiz)
- 009-011: Twilight Zone (Intro, Explained, Quiz)
- 012-014: Midnight Zone (Intro, Explained, Quiz)
- 015-017: Abyss Zone (Intro, Explained, Quiz)
- 018-020: Coral Reefs (Intro, Explained, Quiz)
- 021-023: Marine Mammals (Intro, Explained, Quiz)
- 024-026: Fish (Intro, Explained, Quiz)
- 027-028: Conclusion (Conclusion, Congratulations)

## Dinosaur Discovery

**Author:** Jimmy Cooper  
**Target Age:** 7-8 years  
**Status:** In Development

An interactive journey through prehistoric times, covering:
- Dinosaur eras and periods
- Famous dinosaurs and their characteristics
- Fossils and paleontology
- Extinction theories
- Prehistoric plants and environments

**Features:**
- Standalone HTML files with embedded CSS and JavaScript
- Interactive rotate activities for dinosaur skeletons
- Dig for fossils mini-games
- Wizard-style character selection (user name + avatar)
- Random quiz types: multiple choice, matching, drag-drop, click
- Olympic-style podium celebration after each quiz
- Final overall quiz session after conclusion
