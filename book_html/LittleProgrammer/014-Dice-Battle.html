📖 What are Variables & Random Numbers?

A variable is a box that stores information, like a score. It can change as the program runs.

Random numbers let the computer "roll the dice" - a different number every time, just like a real die!

💻 JavaScript Code

let playerScore = 0; let computerScore = 0; let playerRoll = Math.floor(Math.random() * 6) + 1; let computerRoll = Math.floor(Math.random() * 6) + 1; if (playerRoll > computerRoll) playerScore++;
🌟 Did you know?

Math.random() gives the computer a random decimal between 0 and 1. Multiplying and rounding it turns it into a random whole number, like a dice roll from 1 to 6!
