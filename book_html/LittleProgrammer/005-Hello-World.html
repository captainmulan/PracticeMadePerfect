📖 Order Matters!

A program is a list of instructions that run in order, one after another. This order is called a sequence.

If you mix up the order, you get a mixed-up result - even if you use the exact same instructions!

console.log("Hello"); console.log("World"); console.log("!"); // prints, in this exact order: // Hello // World // !
🌟 Did you know?

"Hello, World!" is the very first program that most programmers ever write. It's a tradition! It teaches that a computer runs your instructions in the exact sequence you write them.
