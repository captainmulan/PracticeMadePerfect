📖 What is If / Compare?

An if statement lets the computer make a decision. It compares two things and asks: "Are these the same?"

If the answer is yes, the computer does one thing. If not, it does something else!

💻 JavaScript Code

if (firstCard.symbol === secondCard.symbol) { // It's a match! matches++; } else { // Not a match, flip back }
🌟 Did you know?

The symbol === means "is exactly equal to." Programmers use it inside if statements to compare two values before deciding what to do next.
