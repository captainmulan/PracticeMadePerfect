📖 What is an Event?

An event is something that happens - like a click!

The computer waits quietly until you click, then it runs your code. That's called "listening" for an event.

💻 JavaScript Code

hole.addEventListener("click", () => { if (hole.classList.contains("up")) { score++; } });
🌟 Did you know?

Every time you click a button, tap a screen, or press a key, you are triggering an event. Programmers write code that says "when this event happens, do this!"
