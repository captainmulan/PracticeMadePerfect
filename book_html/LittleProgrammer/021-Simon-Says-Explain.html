How Simon Says Works

Let's see what happens inside the computer!

💻 Our Program

for (let i = 0; i < sequence.length; i++) { lightUp(sequence[i]); await wait(500); }
1

➕ A New Color Joins

Each round, the computer picks a random color and adds it to the end of the sequence list.

⬇️
2

🔁 The Loop Starts

A for loop goes through the sequence list one color at a time, from the very first to the very last.

⬇️
3

💡 Each Color Lights Up

On every loop step, the computer lights up one pad, waits a moment, then moves to the next.

⬇️
4

🎮 You Repeat It

Now it's your turn to click the same colors, in the same order, using your memory.

⬇️
5

🔂 The Loop Gets Longer

Every round adds one more color, so the loop repeats one extra time. The game keeps growing!
