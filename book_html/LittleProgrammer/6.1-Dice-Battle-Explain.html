How Dice Battle Works

Let's see what happens inside the computer!

💻 Our Program

let playerScore = 0; let playerRoll = Math.floor(Math.random() * 6) + 1; let computerRoll = Math.floor(Math.random() * 6) + 1; if (playerRoll > computerRoll) playerScore++;
1

📦 Variables Are Created

playerScore and computerScore are boxes that start at 0 and will hold a number that can change.

⬇️
2

🎲 The Computer "Rolls"

Math.random() picks a random decimal, and the code turns it into a whole number from 1 to 6 - just like a real die!

⬇️
3

⚖️ The Numbers Are Compared

The program checks: is your roll bigger than the computer's roll?

⬇️
4

➕ A Variable Changes

If you win the round, playerScore goes up by one. The box now holds a new number!

⬇️
5

🔁 It Happens Again

Roll after roll, the variables keep changing until someone wins the battle.
