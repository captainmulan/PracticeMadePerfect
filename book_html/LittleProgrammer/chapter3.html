Chapter 3 — Events & Input

Events are things that happen (a key press, a click, a timer). Programs can wait for events and then run instructions when the event occurs.

What you'll learn

  • How to make a character move when someone presses a key
  • How buttons and clicks can start actions

Activity

Try the demo in Chapter 3.1 to make a sprite respond to keypresses or clicks. You can also try similar ideas in Scratch or Blockly.
