Maze Walker - Explained

Chapter 04 · Commands (Move Up/Down/Left/Right)
CONCEPT 1

🎮 What is a Command?

A command is a single, simple instruction that tells the computer to do one exact thing right now - like "move up!"

Unlike a whole program with many steps, a command usually does just one job, and the computer does it immediately.

CONCEPT 2

📍 Commands Change Coordinates

Every character in a maze has a position made of two numbers: a row and a column. Each command changes one of those numbers:

  • ⬆️ Up → row goes down by 1
  • ⬇️ Down → row goes up by 1
  • ⬅️ Left → column goes down by 1
  • ➡️ Right → column goes up by 1
if (dir === "up") row = row - 1; if (dir === "down") row = row + 1; if (dir === "left") col = col - 1; if (dir === "right") col = col + 1;
row: 2, col: 2
CONCEPT 3

🧱 What About Walls?

Before moving, the computer checks: "is there a wall at the new spot?" If yes, the command is ignored - the character stays put. If no, the move happens.

1

Get the command (up, down, left, or right).

2

Figure out the new row/column that command would move to.

3

Check if that spot is a wall or off the edge of the maze.

4

If it's clear, move there. If not, do nothing.
